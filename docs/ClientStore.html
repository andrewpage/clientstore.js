Class: ClientStore

ClientStore

new ClientStore(options)

ClientStore is a standard interface for storing client side data. It's standard methods (set, get) are defined at runtime based on the availability of certain client-side storage libraries (WebStorage). ClientStore also includes functionality for expiration of WebStorage entries, which is not supported by default by the WebStorage standard.
